# Pivot Reset

A Pivot Reset occurs when a stock breaks out, fails, consolidates again, and forms a new valid pivot.

## How It Works

1. Breakout → failure
2. Stop out
3. Stock consolidates instead of collapsing
4. New tighter base forms
5. New pivot emerges
6. Second breakout succeeds

## Example

> "Craft Brewers Alliance (HOOK) 2010 — got stopped out. The stock reset and then rose 90 percent."
