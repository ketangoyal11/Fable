# Livermore Pivot System

The Livermore Pivot System, used by Jesse Livermore, identifies key pivot points for entry levels.

## Key Principle

> "Livermore never bought at the low. He bought as the price moved up through pivot points."

## Modern Application

Minervini's SEPA strategy is a direct descendant: buy as price emerges through a pivot on volume, with a stop below the danger point.
