# Primary Base

A Primary Base is the first significant consolidation after a stock goes public. It takes time to develop properly.

## Key Point

> "Newly public stocks need time to forge a proper base."

IPO bases that are too short or too wide often fail. The best IPO setups form a primary base over several weeks to months before breaking out.
