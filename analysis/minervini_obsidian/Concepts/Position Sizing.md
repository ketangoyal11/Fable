# Position Sizing

Position Sizing determines capital allocation per trade based on setup quality, market environment, batting average, and portfolio heat.

## Guidelines

- Tight VCPs = full size
- Extended/loose setups = half size
- Hot streak = larger size
- Cold streak = smaller size
- Max 1-2% risk per trade
