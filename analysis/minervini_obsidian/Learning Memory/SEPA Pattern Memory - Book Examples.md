# SEPA Pattern Memory - Book Examples

## Durable Rules Learned

- Leadership is only the first filter; the entry still needs a controllable pivot and nearby invalidation.
- Tightness is best treated as a measurable contraction in volatility, close range, and downside behavior.
- Volume dry-up improves the setup only when it appears near a defined pivot, not randomly in a downtrend.
- Cheat entries are useful only when they reduce risk before the obvious pivot; early buying without tightness is not a cheat entry.
- IPO bases become interesting after supply is absorbed and a first real structure forms.
- A failed pivot is not a stubborn-hold situation; it becomes evidence that the setup was wrong or too early.
